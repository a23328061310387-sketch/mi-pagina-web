Ciberseguridad

Tipos de Malware

Tipo de Malware Función Impacto
Virus Autoreplicable, infecta archivos Daño de datos y del sistema
Gusanos Se propaga a través de las redes Interrupción, agotamiento de recursos
Caballo de Troya Se disfraza de software legítimo Robo de datos, control del sistema
Ransomware Cifra archivos y exige pago Pérdida de acceso a datos
Software espía Supervisa en secreto la actividad del usuario Violación de privacidad
Adware Muestra anuncios no deseados Molestia, bajo rendimiento
Rootkit Oculta software malicioso Acceso no autorizado
Keylogger Registra pulsaciones de teclas Robo de credenciales
Fileless Malware Se ejecuta en memoria Ataques sigilosos y difíciles de detectar
Cryptojacker Usa recursos del sistema para minar criptomonedas Bajo rendimiento, sobrecarga de CPU
Botnet Crea una red de dispositivos infectados Ataques DDoS, envío de spam
Bomba lógica Ejecuta código en un disparador Destrucción de datos
Wiper Destruye datos Pérdida irreversible de información

Ataques de Ciberseguridad

Ataque Descripción Impacto Ejemplo Imagen
Phishing Engaña al usuario para robar credenciales mediante correos falsos. Robo de identidad, pérdida de datos. Correo falso de un banco solicitando datos personales.
SQL Injection Inserta código malicioso en consultas SQL para acceder a bases de datos. Exfiltración y manipulación de información. Campo de login vulnerable permite acceso a datos de usuarios.
Ataque DDoS Satura un servidor con múltiples peticiones para dejarlo inaccesible. Interrupción de servicio, pérdidas económicas. Botnet enviando tráfico masivo a un sitio web hasta que colapsa.
Ransomware Cifra archivos y exige un pago para liberarlos. Pérdida de datos y dinero. Ataques como WannaCry y Petya.
Malware Software malicioso diseñado para dañar, interrumpir o robar información. Pérdida de información, control remoto del equipo. Instalación de spyware o troyanos en segundo plano.
Man-in-the-Middle (MitM) Intercepta la comunicación entre dos partes para robar o modificar información. Robo de credenciales, espionaje. Interceptar datos entre un usuario y un sitio web no cifrado.
Cross-Site Scripting (XSS) Inyección de scripts maliciosos en sitios web que se ejecutan en el navegador. Robo de cookies, credenciales y manipulación de usuarios. Script inyectado en un formulario de comentarios.
Brute Force Attack Prueba todas las combinaciones posibles de contraseñas hasta encontrar la correcta. Acceso no autorizado a cuentas. Intentos automáticos de login con combinaciones de contraseñas.
Zero-Day Attack Explotación de vulnerabilidades desconocidas antes de que se publiquen parches. Acceso total al sistema, difícil de detectar. Uso de exploits antes del parche por el proveedor.
Keylogger Programa que registra cada pulsación de teclado. Robo de contraseñas, mensajes y datos bancarios. Instalación de keylogger en ciber cafés o PCs públicos.
Ingeniería Social Manipulación psicológica para obtener información sensible. Robo de identidad, acceso a sistemas. Llamada fingiendo ser soporte técnico para obtener contraseñas.
Supply Chain Attack Compromete software o hardware en la cadena de suministro. Compromete a miles de empresas con una sola intrusión. Malware insertado en una actualización legítima de software.
